# Student Progress Dashboard

A responsive dashboard to track student academic progress.

## Built With
- Vite + React
- Tailwind CSS
- Framer Motion

## Features
- Responsive grid of student cards
- Sort by Name or Score
- Filter by status (Excellent, On Track, Needs Attention, At Risk)
- Dark / Light mode toggle
- Smooth animations using Framer Motion

## How to Run

1. Clone the repository
2. Install dependencies:
   npm install
3. Start the development server:
   npm run dev
4. Open http://localhost:5173 in your browser

## Mock API
Student data is generated from a local array simulating an API fetch, including name, score, attendance, and status.
